# Tuning

This directory stores the committed summaries for the radius-separation campaign in the active hybrid track.

## Current Campaign

- `radius_separation/`
  - staged hyperparameter sweep for improving depth-by-radius separation

## What Is Committed

- stage-level Markdown and JSON summaries
- baseline calibration notes

## What Is Generated Locally

Not committed:

- full per-run artifact directories
- tuning checkpoints
- exported embedding tables
- per-run plots
- logs

Those are regenerated by the campaign script and intentionally ignored by git.

## Entry Point

Use:

- `../scripts/run_radius_tuning_campaign.py`

to continue the campaign.
