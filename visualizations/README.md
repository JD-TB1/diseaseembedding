# Visualizations

This directory keeps only the small curated hierarchy views that are referenced by the docs.

Current committed files:

- `disease_subtree_90_focus_paths.svg`
- `disease_subtree_1150_leftright_by_route.svg`
- `disease_subtree_90_radial_by_route.svg`

These are reusable orientation figures, not a dump of every generated visualization.

## Generator

Use:

- `../tools/visualize_disease_tree.py`

to regenerate subtree or full-forest views locally.

## Output Policy

Not committed:

- full HTML exports
- poster-only visual bundles
- alternate temporary render variants

Those outputs are generated locally and ignored by git unless they are deliberately promoted into the curated set above.
